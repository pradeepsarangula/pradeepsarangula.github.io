Install Docker Desktop on Windows

Posted on 2021.02.12

Download

  1. Download the Docker Desktop Installer from Docker Hub.

     Click on Get Docker Desktop for Windows

Install

  1. Double-click Docker for Windows Installer to run the installer.

    When the installation finishes, Docker starts automatically. The whale [image_tag] in the notification area indicates that Docker is running, and accessible from a terminal.

Run

  1. Open a command-line terminal like PowerShell, and try out some Docker commands!

     ## Check the docker version
 docker version

 ## Verify the Docker can pull and run images
 docker run hello-world

Enjoy

  1. Docker is available in any terminal as long as the Docker Desktop for Windows app is running. Settings are available on the UI, accessible from the Docker whale in the taskbar.

Documentation

To learn more, read the Docker Desktop for Windows documentation.
